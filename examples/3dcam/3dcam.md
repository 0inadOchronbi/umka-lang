# Build instructions

## Windows (MinGW)

* Copy the following files to the current folder:
  - From Raylib distribution:
    - `raylib.dll`
    - `libraylibdll.a`
    - `raylib.h`
  - From MinGW distribution
    - `libwinpthead-1.dll`
* Add MinGW paths to the `PATH` environment variable
* Run `build_3dcam_windows_mingw.bat`
* Run `3dcam.exe`
